

# TikTok Api：音乐趋势

### 请求Api
```http
http://主机地址/tiktok/sound/trending?token=xxx
```

### 请求方式
```http
GET
```
### 参数
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| token | string | 接口授权码 |
