

# TikTok Api：视频详情（通过链接查询）

### 请求Api
```http
http://主机地址/tiktok/getbyUrl?url=https%3A%2F%2Fwww.tiktok.com%2F%40gordonramsayofficial%2Fvideo%2F6832324357851778309&token=xxx
```

### 请求方式
```http
GET
```
### 参数
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| token | string | 接口授权码 |
| url | string | 视频链接 |
